'Inline' and 'inline-block values', for the CSS3 property, 'display: ;'

Basically, every single website you visit is made up of lots of different boxes, and you are able to control the shape, size and how the boxes interact with the other boxes on the webpage, by changing different properties on the boxes. Sort of like being able to change people’s heights, hair colour and social skills. One property available in CSS is, display.

The display property changes how the boxes in a website interact with other boxes around it. Two different values for display are, ‘Inline’ and ‘inline-block’. The major difference between them is that inline-block will push other boxes, away from it so that it doesn’t overlap with any other boxes, while with inline, the box will overlap with other elements around it.

Imagine you are drawing a scorecard for a game of cards; you’ve written down the player names and now want to drawn the vertical lines that makes up the columns so that they are evenly spread out 5cm apart from each other. Unfortunately you have big handwriting and you’ve written Jo$eph down so that it takes up more than 5cm of space, If you were to draw the scorecard around the names with the ‘display: inline;’ property, you would draw all the columns in at 5cms and you would draw the line right through Jo$eph

If you were to draw in a scorecard around the names with the display: inline-block; property you would draw most of the columns at 5cm widths but when it comes round to drawing the line between Jo$eph and Oliver you allow for the extra big handwriting and draw the column bigger than 5cms
